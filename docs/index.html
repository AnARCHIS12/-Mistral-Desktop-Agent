Installation

Demarrage rapide

Debutants

Clone le depot ou telecharge le ZIP, puis lance le menu adapte a ton systeme.

git clone https://github.com/AnARCHIS12/-Mistral-Desktop-Agent.git Mistral-Desktop-Agent
cd Mistral-Desktop-Agent

Windows

Double-clique le fichier de menu.

setup_windows.bat

Linux

Lance le menu ou cree un raccourci bureau.

bash launch.sh
bash create_desktop_launcher.sh

Expert

Installation directe depuis GitHub avec curl.

curl -fsSL https://raw.githubusercontent.com/AnARCHIS12/-Mistral-Desktop-Agent/main/bootstrap.sh | bash

Fonctions

Controle agentique complet

Interface web

Objectif, start/stop, logs live, action courante et capture ecran en temps reel.

Boucle autonome

Screenshot, OCR, prompt Mistral, action JSON stricte, execution, memoire SQLite.

Outils locaux

Terminal, fichiers, souris, clavier, navigateur Playwright et vision ecran.

Telegram

Commandes /goal, /start, /stop et /status pour piloter l'agent a distance.
